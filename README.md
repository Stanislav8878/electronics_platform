Electronics Platform
Описание проекта

Веб-приложение для управления сетью по продаже электроники, реализованное с использованием Django и Django REST Framework.

Приложение моделирует иерархическую структуру сети поставщиков электроники и предоставляет:

административную панель;

REST API;

фильтрацию и разграничение доступа;

автоматическое тестирование.

Функциональность
Иерархическая модель сети

Сеть состоит из трёх уровней:

Завод (уровень 0)

Розничная сеть (уровень 1)

Индивидуальный предприниматель (уровень 2)

Особенности:

каждое звено имеет только одного поставщика;

уровень определяется положением в иерархии;

запрещены циклы и некорректные связи.

Поля модели

название;

email;

страна;

город;

улица;

номер дома;

поставщик;

задолженность (Decimal);

дата создания.

Продукты

название;

модель;

дата выхода на рынок.

Ограничения

завод не может иметь поставщика;

запрещена ссылка на самого себя;

запрещены циклические зависимости;

максимум 3 уровня иерархии.

Админ-панель

Реализовано:

просмотр объектов сети;

ссылка на поставщика;

отображение уровня;

фильтр по городу;

admin action — очистка задолженности.

API
CRUD
GET     /api/nodes/
POST    /api/nodes/
GET     /api/nodes/{id}/
PATCH   /api/nodes/{id}/
DELETE  /api/nodes/{id}/
Особенности

поле debt доступно только для чтения;

фильтрация по стране:

/api/nodes/?country=Germany
Документация API

Swagger: http://127.0.0.1:8000/api/docs/

ReDoc: http://127.0.0.1:8000/api/redoc/

Schema: http://127.0.0.1:8000/api/schema/

Права доступа

Тестирование

Запуск тестов:

Результат:

Покрытие тестами (Coverage)

Для расчёта покрытия используется библиотека coverage.

Запуск:
docker compose exec web coverage run manage.py test
Отчёт:
docker compose exec web coverage report

Пример:

TOTAL ... 90%+
